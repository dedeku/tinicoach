## 9. PROFIL & BEÁLLÍTÁSOK

### 9.1 Profil Megtekintése

**Output:**

- Név
- Email
- Születési dátum
- Subscription plan
- Member since

### 9.2 Profil Szerkesztése

**Input:**

- Új név
- Új jelszó (optional)

**CRUD:**

```sql
UPDATE User
SET name = ?, password_hash = ?
WHERE id = ?
```

### 9.3 Beállítások

**Toggles:**

- Push notifications (on/off)
- Email notifications (on/off)
- Dark mode (on/off) (optional MVP)

**CRUD:**

```sql
UPDATE UserSettings
SET push_enabled, email_enabled, dark_mode
WHERE user_id = ?
```

### 9.4 Adatok Exportálása (GDPR)

**Button:** "Adataim letöltése"

**Output:**

- JSON file download
- Tartalom: összes napló, szokás, kvíz eredmény

### 9.5 Fiók Törlése (GDPR)

**Flow:**

- Confirmation modal: "Biztosan törlöd?"
- Input: "TÖRLÉS" szó beírása
- Delete user + all related data (cascade)

**CRUD:**

```sql
DELETE User CASCADE
WHERE id = ?
```

## 1. AUTHENTICATION & USER MANAGEMENT

### 1.1 Regisztráció

**Input:**

- Email cím
- Jelszó (min 8 karakter)
- Név
- Születési dátum
- Szülő email címe (ha <16 éves)

**Validáció:**

- Email formátum ellenőrzés
- Életkor számítás születési dátumból
- Ha <16 éves → szülői email KÖTELEZŐ

**Output:**

- User account létrehozva
- Activation email küldése
- Ha <16 éves → Parent consent email küldése

**CRUD:**

```sql
CREATE User {
  id, email, password_hash, name, birth_date,
  parent_email?, parent_consent_given (boolean),
  subscription_plan ('FREE' default), created_at
}
```

### 1.2 Bejelentkezés

**Input:**

- Email
- Jelszó

**Output:**

- JWT token / Session
- Redirect to Dashboard

**Edge cases:**

- Email not verified → "Kérlek erősítsd meg az email címed"
- Parent consent pending (<16) → "Várjuk a szülői jóváhagyást"

### 1.3 Jelszó Reset

**Flow:**

- Input: Email
- Send reset link
- Input: New password
- Update password_hash

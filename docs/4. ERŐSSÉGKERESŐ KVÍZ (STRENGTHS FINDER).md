## 4. ERŐSSÉGKERESŐ KVÍZ (STRENGTHS FINDER)

### 4.1 Kvíz Struktúra

> **Megjegyzés**: CMS integráció TBD (lásd: MVP PRIORITÁS.md). Jelenleg a kvíz kérdések hardcoded vagy adatbázisban tárolva.

**Tervezett Content Type (CMS-ben):**

**Content Type: StrengthQuiz**

- questions[] (20-30 kérdés)
    - question_text
    - strength_category (pl: "Kreativitás", "Bátorság", "Igazságosság")

### 4.2 Kvíz Kitöltése

**Input:**

- 20-30 kérdés, mindegyikhez:
    - Likert scale: 1-5 (egyáltalán nem jellemző → nagyon jellemző)

**Flow:**

- Progressbar (1/20, 2/20...)
- "Vissza" és "Következő" gombok
- Save progress (ha félbehagyja)

**CRUD:**

```sql
CREATE QuizProgress {
  id, user_id, quiz_id,
  current_question_index,
  answers (JSON: {question_id: score}),
  completed (boolean),
  created_at, updated_at
}
```

### 4.3 Kvíz Kiértékelés

**Logic:**

- Összegzés strength_category szerint
- Top 5 kategória kiszámítása
- Leírások generálása

**CRUD:**

```sql
CREATE QuizResult {
  id, user_id, quiz_id,
  top_strengths (JSON: [{name, score, description}]),
  completed_at
}

UPDATE QuizProgress
SET completed = true
WHERE id = ?
```

**Output:**

- "Az erősségeid" screen
- Top 5 erősség leírással
- "Mentés" és "Újrakezdés" gombok

### 4.4 Erősségek Megtekintése

**Input:** User ID

**Output:**

- Utolsó kvíz eredménye
- Ha nincs: "Töltsd ki a kvízt!"

**Feature Gate:**

- Ingyenes: 1x kitöltheti
- Premium: korlátlan
